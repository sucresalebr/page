# Sucré & Salé - GitHub Pages

Projeto adaptado a partir do template Sarab, mantendo a estrutura visual do modelo e substituindo conteúdo, cores e imagens pela identidade da Sucré & Salé.

## Publicação

1. Envie estes arquivos para um repositório no GitHub.
2. Em **Settings > Pages**, selecione a branch principal e a pasta raiz.
3. Aguarde a publicação do GitHub Pages.

## Personalização rápida

No arquivo `index.html`, substitua os placeholders:

- `https://wa.me/5500000000000`
- `contato@sucreesale.com.br`
- link do Instagram
